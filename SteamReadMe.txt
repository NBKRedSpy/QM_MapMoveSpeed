[h1]Quasimorph MapMoveSpeed[/h1]


Holding down the shift key while in a raid will speed up the scrolling of the map.

Key and speed can be defined in the config file.

[h1]Configuration[/h1]

The configuration file will be created on the first game run and can be found at [i]%AppData%\..\LocalLow\Magnum Scriptum Ltd\Quasimorph_ModConfigs\MapMoveSpeed\config.json[/i].
[table]
[tr]
[td]Name
[/td]
[td]Default
[/td]
[td]Description
[/td]
[/tr]
[tr]
[td]BorderMoveSpeed
[/td]
[td]10
[/td]
[td]The speed of the map movement.  The game defaults to 4.  Higher is faster.
[/td]
[/tr]
[tr]
[td]SpeedKey
[/td]
[td]LeftShift
[/td]
[td]The key used to enable the speed change
[/td]
[/tr]
[/table]

[h2]Key List[/h2]

The list of valid keyboard keys can be found  at the bottom of https://docs.unity3d.com/ScriptReference/KeyCode.html
Beware that numbers 0-9 are Alpha0 - Alpha9.  Most of the other keys are as expected such as X for X.
Use "None" to not bind the key.
